Django-Celery-Redis项目示例
=====
本项目是celery的简单演示实例，利用celery实现异步任务(已实现)和定时任务(开发中)。<br>
本示例只有代码，还需大家安装相关的支持库

测试环境
-----
Python版本和相关支持库的版本<br>
* Python 2.7 (代码已经兼容py2和py3)
* Django 1.10
* celery 3.1.25
* redis 2.10.5
* djano-redis 4.7.0
* celery-with-redis 3.0

Redis数据库的版本
* redis-3.2.8
